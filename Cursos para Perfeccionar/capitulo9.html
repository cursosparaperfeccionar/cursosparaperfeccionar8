Aspectos Claves

Para reproducir sonido disponemos de la etiqueta audio. En su versión mínima sólo necesita una fuente de sonido y un control de reproducción.

Esta etiqueta audio es muy similar a la que acabamos de ver para vídeo, si no estamos seguros de que el navegador vaya a poder reproducir el formato de vídeo podemos incluir etiquetas source con otros formatos, también podemos incluir el atributo type con el tipo de formato de audio:

Con funciones similares a la etiqueta vídeo, disponemos de los atributos autoplay, controls, mute, loop y preload. El tipo de vídeo mpeg (.mp3) es claramente el más utilizado en la web y es reproducido por todos los navegadores, otros tipos estándar de HTML 5 son ogg y wav. Existen otros tipos de audio como el .m4a que utiliza la grabadora de voz de Windows.
